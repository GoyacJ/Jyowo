# 可调整侧栏分割线实施计划

## 目标

实现可拖动、可键盘调整、可收起和恢复的侧栏分割控件，并持久化展开宽度。

## 实施步骤

### 1. 侧栏宽度状态

- 增加侧栏宽度常量和限制函数。
- 在 UI store 中增加 `sidebarWidth` 与更新操作。
- 在本地 UI 偏好中读取、校验和写入宽度。

### 2. 分割控件

- 在应用壳层中按状态计算 48px 或展开宽度。
- 增加覆盖在边界上的 8px 拖动区域和 1px 可见线。
- 增加中点收起/展开按钮、指针拖动和键盘调整。
- 移除侧栏顶部原按钮，保留原顶部占位。

### 3. 验证

- 覆盖默认宽度、拖动限制、收起恢复、窄屏和键盘操作。
- 覆盖 UI store 与本地偏好读写。
- 运行 `pnpm check:frontend:fast` 和 `pnpm check:quick`。
