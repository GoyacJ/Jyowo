# 可调整侧栏分割线设计

## 目标

将工作区侧栏右侧的静态边框改为复合分割控件：展开时可拖动调整宽度，悬浮或获得焦点时在垂直中点显示收起按钮；收起后保留 48px 图标栏，并在同一位置提供展开按钮。

## 核心决策

- 展开宽度限制为 240–420px，默认 300px。
- 收起宽度固定为 48px；再次展开恢复收起前的宽度。
- 分割线可见宽度为 1px，鼠标命中宽度为 8px。
- 按钮与拖动区域分离，避免点击和拖动互相触发。
- 窄于或等于 720px 时沿用现有紧凑侧栏，不开放拖动和分割线按钮。
- 删除侧栏顶部按钮，但保留 48px 顶部布局区域。
- 宽度与收起状态都作为本地 UI 偏好保存。

## 状态

| 状态 | 分割线 | 中点按钮 | 宽度 |
|---|---|---|---|
| 展开、静止 | 弱边框 | 隐藏 | 上次展开宽度 |
| 展开、悬浮或聚焦 | 高亮 | 收起 | 上次展开宽度 |
| 拖动 | 高亮 | 隐藏 | 240–420px |
| 收起 | 弱边框 | 悬浮或聚焦时显示展开 | 48px |
| 窄屏 | 现有侧栏边框 | 不显示 | 48px |

## 无障碍

- 拖动区域使用垂直 `separator` 语义并公开当前值、最小值和最大值。
- 展开状态下支持方向键按 10px 调整，Home/End 跳到最小值和最大值。
- 收起和展开按钮继续使用中英文可访问名称。
- 按钮在键盘焦点进入时显示，不只依赖鼠标悬浮。

## 非目标

- 不改变侧栏内容结构。
- 不改变 48px 紧凑模式的菜单表现。
- 不增加双击复位、吸附档位或完全隐藏到 0px 的模式。
