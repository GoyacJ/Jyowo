# MCP 2025-11-25 Remediation Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Make Jyowo's MCP client, runtime integration, desktop configuration, optional transports, and reverse server adapter conform to MCP 2025-11-25 while preserving task isolation and permission boundaries.

**Architecture:** Introduce shared typed protocol messages, a bidirectional peer, and negotiated session state above transport-specific codecs. Keep settings, foreground, and child registries isolated, but unify configuration semantics, live state, change propagation, diagnostics, and trust derivation.

**Tech Stack:** Rust, Tokio, Reqwest, Axum, tokio-tungstenite, Serde, React, TypeScript, Zod, Tauri, Vitest, Cargo.
