# 安全工作区目录适配器实施计划

## 1. 安全目录遍历

- 为 `WorkspaceToolAuthorization` 增加只读目录访问选项、条目类型和回调接口。
- 使用目录文件描述符递归打开后代目录和普通文件。
- 跳过符号链接与特殊文件，并限制深度、条目数和单文件读取大小。

## 2. 工具适配器

- 为 `ListDir` 生成相对路径、类型、大小和修改时间。
- 为 `Glob` 在安全枚举结果上执行 glob 匹配。
- 为 `Grep` 逐文件读取 UTF-8 内容并执行正则匹配。
- 未实现适配器的文件系统工具继续拒绝。

## 3. 验证

- 扩展 daemon 工具注册测试，覆盖三个工具的成功执行和既有输出语义。
- 覆盖隐藏文件、递归深度、排序、无效正则和符号链接逃逸。
- 运行相关 Rust 测试、`pnpm check:rust:fast` 和 `pnpm check:quick`。
